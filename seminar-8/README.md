# Семинар 8: Хранимые процедуры и триггеры

## Описание

В этом задании вам предстоит создать процедуры, функции и триггеры на языке PL/pgSQL для базы данных сотрудников компании. Работа выполняется с использованием PostgreSQL.

## Инструкция по выполнению

1. Клонируйте ваш репозиторий, созданный GitHub Classroom.
2. Запустите контейнер PostgreSQL с помощью файла `docker-compose.yml`, расположенного в корне проекта. База импортируется автоматически.
3. Выполните задание в файле `./src/answers.md`.
4. Сделайте коммит и пуш изменений в ваш репозиторий.
