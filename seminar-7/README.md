# Семинар 7: ORM.

## Описание

В этом задании вы будете работать с базой данных, моделирующей Олимпийские игры, используя ORM на Python — рекомендуется
SQLAlchemy, но вы можете выбрать любой ORM-фреймворк (например, Tortoise ORM, Pony ORM).
Вы создадите модели сущностей, выполните запросы к базе данных и обработаете результаты.

## Инструкция по выполнению

1. Клонируйте ваш репозиторий, созданный GitHub Classroom.
2. Запустите Docker-контейнеры с базой данных и миграцией.
3. Создайте ORM-модели для всех таблиц и реализуйте SQL-запросы из задания, используя выбранный ORM. Рабочие файлы
   находятся в каталоге `/src` — туда и пишите код.
4. Реализуйте SQL-запросы с использованием ORM в файле `main.go` в каталоге `/src`.
5. Запустите повторно контейнер с сервисом приложения, чтобы увидеть результаты выполнения запросов -
   `docker compose up app`.
6. После завершения задания создайте commit и отправьте изменения в репозиторий.

## Описание данных

Ваша база данных моделирует Олимпийские игры. У вас есть следующие сущности:

- **Countries** (страны-участники)
- **Olympics** (Олимпиады: год, страна, город, даты проведения)
- **Players** (спортсмены: имя, страна, дата рождения)
- **Events** (соревнования: название, тип, команда/индивидуальное, олимпийские игры)
- **Results** (результаты: спортсмен, соревнование, медаль, баллы)

- **countries** (country_id, name, area_sqkm, population)
- **olympics** (olympic_id, country_id, city, year, startdate, enddate)
- **players** (player_id, name, country_id, birthdate)
- **events** (event_id, name, eventtype, olympic_id, is_team_event, num_players_in_team, result_noted_in)
- **results** (event_id, player_id, medal, result)

## Запросы с использованием ORM

Реализуйте следующие SQL-запросы, используя ORM:

1. **Для Олимпийских игр 2004 года** найдите (год рождения, количество игроков, количество золотых медалей), где:
    - `год рождения` – это год рождения спортсменов
    - `количество игроков` – число спортсменов, родившихся в этот год
    - `количество золотых медалей` – общее число золотых медалей у спортсменов этого года рождения

2. Перечислите **все индивидуальные соревнования**, в которых была ничья, и два или более игрока выиграли золотую
   медаль.

3. Найдите **всех игроков, которые выиграли хотя бы одну медаль** (GOLD, SILVER, BRONZE) на одной Олимпиаде. Выведите
   `player_name` и `olympic_id`.

4. В какой **стране** был **наибольший процент игроков**, чьи имена начинались с гласной?

5. Для Олимпийских игр **2000 года** найдите **5 стран с минимальным соотношением количества групповых медалей к
   численности населения**.

## Формат сдачи работы

- Все Python-файлы должны быть размещены в каталоге /src. Точка входа — файл main.py.
- **Миграции** – уже готовы, находятся в папке migration, запускаются при старте контейнера.
- Модели — в любом файле/пакете внутри /src.
- выполнение запросов — обязательно вызывается из main.py.
- **Docker-compose для запуска базы данных** – в корне, запускать командой `docker compose up -d`.
- После выполнения задания, зафиксируйте изменения (`git commit -m "Seminar 7 completed"`).
- Отправьте их в ваш репозиторий (`git push origin main`).
